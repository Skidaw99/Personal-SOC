Privacy Policy

Last updated: April 10, 2026

1 About This Application

Security Monitoring Engine ("the App") is an internal security tool operated by PersonalSOC for the purpose of monitoring authorized social media accounts against unauthorized access, fraud, and account takeover attempts.

This App is NOT a public-facing consumer application. It is used exclusively by its operator for internal security monitoring purposes.

2 Information We Collect

Via Facebook Login, we collect:

  • Basic profile information (name, email address)
  • Login activity signals and session metadata
  • Connected application permissions status

We do NOT collect:

  • Posts, messages, or private communications
  • Friend lists or social graph data
  • Photos, videos, or media content
  • Financial information

3 How We Use Information

All collected data is used exclusively for:

  • Detecting unauthorized access to our own accounts
  • Identifying suspicious login patterns and geographic anomalies
  • Generating security incident reports
  • Coordinating with law enforcement when criminal activity is detected

We do NOT:

  • Sell data to third parties
  • Use data for advertising or marketing
  • Share data with any external services except as required by law
  • Use data for any purpose other than security monitoring

4 Data Storage and Security

  • All data is stored on private, encrypted infrastructure
  • Access tokens are encrypted at rest using AES-256 (Fernet)
  • Data is retained for 90 days for security audit purposes
  • Access to stored data is restricted to the operator only

5 Facebook Data Deletion

To request deletion of your data collected via Facebook Login, contact us at: security@personalsoc.com

We will process deletion requests within 30 days.

Facebook Data Deletion Callback URL:
https://personalsoc.com/api/auth/facebook/deletion

6 Third-Party Services

This app uses the following services for security intelligence:

  • MaxMind GeoIP2 (IP geolocation)
  • AbuseIPDB (IP reputation)

No Facebook user data is shared with these services.

7 Legal Basis (GDPR)

We process data under legitimate interest (Art. 6(1)(f) GDPR) for fraud prevention and security monitoring of our own systems.

8 Contact

For privacy inquiries: security@personalsoc.com
